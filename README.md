## 🚀 Descripción del Proyecto

Este proyecto simula un sistema de agendamiento de citas para una veterinaria esta siendo desarrollado en **Java SE con Swing**.  
Incluye navegación moderna entre diferentes secciones y funcionalidades CRUD para la gestión de:

- Usuarios

## 📂 Estructura del Proyecto

- **Menú principal de navegación** con logo e ítems:
  - Usuarios
  - (Otros módulos del sistema)

- **Vistas CRUD**
  - Registrar Usuarios
  - Gestión de Usuarios

- **Opción de Salir**
  - Mensaje informativo de cierre de sesión

## 🔧 Instalación y Ejecución

### 🗄️ Base de Datos
Crear la base de datos **veterinaria_db** en PostgreSQL y ejecutar el script SQL adjunto en la carpeta `db`.

Este script incluye:
- Creación de tablas
- Llaves primarias
- Llaves foráneas
- Restricciones de integridad

---

### 📚 Librerías
Verificar que el proyecto tenga configurados los siguientes JAR en el classpath:

- Driver de PostgreSQL
- FlatLaf (interfaz gráfica moderna)
